# AI/ML Scenario-Based Projects

This folder contains 10 scenario-based AI/ML projects demonstrating various NLP and deep learning concepts.

## Projects

### NLP & Text Analysis
1. **sports_ner_analysis.py** - Named Entity Recognition for sports match reports
2. **manufacturing_word2vec.py** - Word embeddings for manufacturing concepts
3. **healthcare_word2vec.py** - Word embeddings for healthcare research
4. **bert_tokenizer_demo.py** - BERT tokenization for research and healthcare

### GPT-2 Text Generation
5. **email_assistant_gpt2.py** - AI-powered email auto-completion
6. **game_storyline_generator.py** - Fantasy RPG quest and dialogue generator
7. **interactive_text_generator.py** - News headlines and essay brainstorming
8. **temperature_creativity_demo.py** - Temperature effects on AI creativity
9. **model_comparison_gpt2.py** - GPT-2 vs DistilGPT-2 comparison

### Chatbots
10. **university_chatbot.py** - AI helpdesk chatbot with intent recognition

## Running the Projects

All projects run without external dependencies (simplified implementations):

```bash
# NLP & Text Analysis
python "ai ml/sports_ner_analysis.py"
python "ai ml/manufacturing_word2vec.py"
python "ai ml/healthcare_word2vec.py"
python "ai ml/bert_tokenizer_demo.py"

# GPT-2 Text Generation
python "ai ml/email_assistant_gpt2.py"
python "ai ml/game_storyline_generator.py"
python "ai ml/interactive_text_generator.py"
python "ai ml/temperature_creativity_demo.py"
python "ai ml/model_comparison_gpt2.py"

# Chatbots
python "ai ml/university_chatbot.py"
```
