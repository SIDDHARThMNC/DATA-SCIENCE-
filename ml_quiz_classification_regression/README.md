# Machine Learning Quiz: Classification or Regression?

## Overview
This quiz helps you understand the fundamental difference between Classification and Regression problems in Machine Learning.

## What's Inside
- **ml_quiz.py** - Interactive quiz with 15 real-world scenarios
- **quiz_data.csv** - All questions with answers and explanations
- **key_differences.csv** - Detailed comparison table

## Key Concepts

### Classification
- Predicts **discrete categories or labels**
- Output is a class membership (e.g., Yes/No, Cat/Dog/Bird)
- Examples: Email spam detection, Disease diagnosis, Image recognition

### Regression
- Predicts **continuous numerical values**
- Output is a quantity or amount (e.g., price, temperature, age)
- Examples: House price prediction, Stock forecasting, Temperature prediction

## How to Run
```bash
python ml_quiz.py
```

2. Understand the key differences between both approaches
3. Learn common algorithms and evaluation metrics for each type
